# Agent instructions

## Git commits

- Never add yourself as a co-author on commits.
- Keep commit messages short: a single subject line is almost always enough.

## Pull requests

- PR descriptions should be extremely terse. No filler, no restating what the diff already shows.

## Writing style

- No spaces around em dashes.
- Prefer alternatives to em dashes when possible (rewrite the sentence, use a colon or parentheses).
